## Evaluatie van predicaten in code
Predicatenlogica komt veel voor in code, vooral bij **filtering, querying en beslislogica**.  
Een predicaat is in feite een **functie die `true` of `false` retourneert** op basis van een invoer.

### Voorbeeld 1 – Predicaten als functies
```csharp
bool IsGeslaagd(Student s) => s.Cijfer >= 5.5;
bool IsAanwezig(Student s) => s.Aanwezigheid >= 80;

var student = new Student { Naam = "Anna", Cijfer = 7.0, Aanwezigheid = 90 };
Console.WriteLine(IsGeslaagd(student));  // true
```

### Voorbeeld 2 – Gebruik van kwantoren met LINQ
LINQ maakt het makkelijk om kwantoren te gebruiken:

```csharp
bool iedereenGeslaagd = studenten.All(s => s.Cijfer >= 5.5); // ∀x Geslaagd(x)
bool iemandGezakt = studenten.Any(s => s.Cijfer < 5.5);      // ∃x ¬Geslaagd(x)
```

> [!tip] Casus
> "Niet alle studenten zijn geslaagd" ⇔ ¬∀x Geslaagd(x) ⇔ ∃x ¬Geslaagd(x)  
> ```csharp
> bool nietIedereenGeslaagd = studenten.Any(s => s.Cijfer < 5.5);
> ```

### Voorbeeld 3 – Complexere combinaties
```csharp
bool goedBezig = studenten.All(s =>
    (s.Cijfer >= 5.5 && s.Aanwezigheid >= 80) ||
    (s.Cijfer >= 7.0 && s.Aanwezigheid >= 70)
);
```

Hier wordt een samengestelde logische regel toegepast met *en* (∧), *of* (∨) en *implicatievorm* in code.

---

## Samenvatting
| Concept | Logisch symbool | LINQ-functie | Voorbeeld |
|----------|----------------|--------------|------------|
| ∀x P(x) | Universeel | `.All()` | `studenten.All(s => s.Cijfer >= 5.5)` |
| ∃x P(x) | Existentiëel | `.Any()` | `studenten.Any(s => s.Cijfer < 5.5)` |
| ¬P(x) | Negatie | `!` of `== false` | `!s.IsActief` |

> Predicatenlogica in code is dus niets anders dan logische filtering toegepast op objecten.
