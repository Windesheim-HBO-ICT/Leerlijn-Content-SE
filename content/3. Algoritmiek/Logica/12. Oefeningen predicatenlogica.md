## Opdracht 1 – Propositie of predicaat?
Bepaal of de volgende uitspraken proposities of predicaten zijn.

```
1. 5 > 3  
2. x > 3  
3. Alle getallen groter dan 5 zijn positief  
4. Er bestaat een getal kleiner dan 0  
```

> [!info]- Uitwerking
> 1. Propositie: waar (5 en 3 zijn ingevuld).  
> 2. Predicaat: variabele x is niet ingevuld.  
> 3. Predicaat: Universele uitspraak: ∀x (x > 5 → x > 0).  
> 4. Predicaat: Existentiële uitspraak: ∃x (x < 0).

---

## Opdracht 2 – Vertaal naar logische notatie
Zet de volgende uitspraken om in predicatenlogica.

```
1. Elke docent geeft minstens één vak.
2. Sommige studenten volgen geen enkel vak.
3. Niet elke student is geslaagd.
4. Er is precies één student met een 10.
```

> [!info]- Mogelijke uitwerking
> 1. ∀x ( Docent(x) → ∃y Geeft(x, y) )  
> 2. ∃x ( Student(x) ∧ ∀y ¬Volgt(x, y) )  
> 3. ¬∀x ( Student(x) → Geslaagd(x) )  
> 4. ∃x ( Student(x) ∧ Cijfer(x) = 10 ∧ ∀y ( y ≠ x → Cijfer(y) ≠ 10 ) )

## Opdracht 3 – Ontken de uitspraken
Ontken de volgende uitspraken en herschrijf ze in correcte vorm.

```
1. ∀x ( Student(x) → Geslaagd(x) )
2. ∃x ( Docent(x) ∧ Slim(x) )
```

> [!info]- Uitwerking
> 1. ¬∀x ( Student(x) → Geslaagd(x) ) ⇔ ∃x ( Student(x) ∧ ¬Geslaagd(x) )  
> 2. ¬∃x ( Docent(x) ∧ Slim(x) ) ⇔ ∀x ( Docent(x) → ¬Slim(x) )
