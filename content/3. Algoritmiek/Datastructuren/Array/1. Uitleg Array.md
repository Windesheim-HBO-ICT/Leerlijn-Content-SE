## Wat is een Array?
Een **array** is een lijst van dezelfde elementen die in een bepaalde
volgorde staan.

Kenmerken van een array:
- Een **array** heeft een vaste lengte
- Een **array** bevat elementen van hetzelfde type
- De volgorde van elementen is belangrijk
- Je kunt elk element opvragen via een **index** (in C# beginnend bij 0)
- **Arrays** kunnen cijfers, tekst of zelfs andere **arrays** bevatten.

## Hoe zit een array in elkaar?
- De elementen van een **array** kun je opvragen en veranderen door
het gebruik van blokhaken rondom de index: `[]`.
- Bij de declaratie van een **array** moet je direct aangeven hoe
groot de array is.
- Je kunt de lengte van een array op vragen met de property `Length`

```csharp
int[] scores = new int[3];   // Array met 3 plekken

scores[0] = 10;
scores[1] = 20;
scores[2] = 30;

Console.WriteLine(scores[1]);  // Output: 20
```

De initialisatie kan ook korter:
```csharp
int[] scores = { 10, 20, 30, 40 };

Console.WriteLine(scores[2]);  // Output: 30
Console.WriteLine(scores.Length);  // Output: 4
```

## Hoe gebruik je [onderwerp]?

> [!TIP] Casus
>Je wil een gebruiker 5 getallen laten invoeren en hier het gemiddelde van berekenen.

**Mogelijke uitwerking van de casus**
```csharp
// 1. Een array aanmaken voor 5 getallen
int[] getallen = new int[5];
int totaal = 0;

// 2. Input van de gebruiker vragen
for (int i = 0; i < getallen.Length; i++)
{
    Console.Write("Voer getal " + (i + 1) + " in: ");
    string invoer = Console.ReadLine();
    getallen[i] = int.Parse(invoer);  // omzetten naar een getal
    totaal += getallen[i];            // optellen bij totaal
}

// 3. Gemiddelde berekenen
double gemiddelde = (double)totaal / getallen.Length;

// 4. Resultaat tonen
Console.WriteLine("Je invoer:");
foreach (int getal in getallen)
{
    Console.WriteLine("- " + getal);
}

Console.WriteLine("Gemiddelde: " + gemiddelde);
```
