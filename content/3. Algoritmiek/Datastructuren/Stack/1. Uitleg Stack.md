## Wat is een Stack?
Een **stack** (Nederlands: **stapel**) is een container waarvan alleen
het laatst toegevoegde element beschikbaar is. Nieuwe elementen worden
bovenop toegevoegd. Dit principe wordt ook wel **LIFO** (Last In First
Out) genoemd.

## Hoe zit een stack in elkaar?


### Kenmerken van een stack
- Een **stack** bevat elementen van hetzelfde type
- Het bovenste element is beschikbaar en kan bekeken of verwijderd
  worden
- Nieuwe elementen worden bovenop de stack toegevoegd

*Figuur 1: Een stack*

### Bewerkingen op een stack
Een **stack** heeft de volgende bewerkingen:
- `Push` : het toevoegen van een element bovenop de **stack**
- `Pop` : het afnemen van het bovenste element van de **stack**
- `Count` : het aantal elementen in de **stack**
- `Peek` : het bekijken van het bovenste element, zonder het eraf te halen
- `Clear` : het leegmaken van de **stack**

![[1.2 stack werking 500px.png]]
*Figuur 1: Een stack*

### Voorbeeld
```csharp
// Maak een stack van strings aan
Stack<string> stack = new Stack<string>();

// Items toevoegen (push)
stack.Push("Eerste");
stack.Push("Tweede");
stack.Push("Derde");

Console.WriteLine("Bovenste item (peek): " + stack.Peek()); // Verwacht: Derde

// Items verwijderen (pop)
Console.WriteLine("Verwijder: " + stack.Pop()); // Verwacht: Derde
Console.WriteLine("Nieuwe bovenste item: " + stack.Peek()); // Verwacht: Tweede

// Stack doorlopen
Console.WriteLine("Overgebleven items in de stack:");
foreach (string item in stack)
{
    Console.WriteLine(item);
}

// Leegmaken
stack.Clear();
Console.WriteLine("Stack is leeg? " + (stack.Count == 0));
```

## Hoe gebruik je een stack?
Dit zijn een aantal toepassingen van een stack:
- Expressie validatie (postfix notatie)
- Valideren haakjes
- Memory management
- Call stack


> [!TIP] Casus
> Draai een string om met behulp van een stack.
> 
> **Mogelijke uitwerking van de casus**
> ```csharp
> string Omkeren(string tekst)
> {
>     Stack<char> stack = new Stack<char>();
>     string resultaat = "";
> 
>     foreach (char teken in tekst)
>     {
>         stack.Push(teken);
>     }
>
>     while (stack.Count > 0)
>     {
>         resultaat += stack.Pop();
>     }
>
>     return resultaat;
> }
> Console.WriteLine(Omkeren("Keer dit om"));
> ```
